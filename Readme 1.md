Clase Vector3D
Descripción

Esta clase implementa un vector tridimensional en C++ con operaciones básicas de álgebra vectorial. Permite crear vectores, calcular su magnitud, normalizarlos y operar entre ellos mediante sobrecarga de operadores.

Características

Representa un vector en el espacio 3D como (x, y, z).

Constructor que inicializa las coordenadas.

Método para calcular la magnitud.

Método para normalizar el vector.

Sobrecarga de operadores:

+ suma vectorial

- resta vectorial

* producto escalar

Método imprimir() para mostrar el vector en consola.

Uso esperado

Ideal para programas de física, computación gráfica, videojuegos o cualquier área donde se usen vectores tridimensionales.
